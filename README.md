# sofiaa_agrinhoo_1A
**Agro Forte, Futuro Sustentável** é um site interativo que destaca a importância do agronegócio aliado à sustentabilidade. Com design moderno, informações educativas, curiosidades, navegação intuitiva e atividades interativas, o projeto mostra como tecnologia, preservação ambiental e produção responsável podem construir um futuro melhor.
